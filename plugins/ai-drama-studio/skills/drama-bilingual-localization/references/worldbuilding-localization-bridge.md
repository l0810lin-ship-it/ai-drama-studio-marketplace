# 世界观到本地化的桥接

本地化必须先读取当前项目的 Project Bible。团队世界观网站与下列类型规则只作为补充参考，不能覆盖项目锁定项。原始资料来自团队本地 `worldbuilding-research` 数据与 `NA-supernatural-worldbuilding-database-v1.md`；公开浏览入口为 `https://na-supernatural-worldbuilding.vercel.app/`。

## 每个项目都要提取的约束

建立一页世界观本地化配置：

- `era_and_place`：年代、国家/城市、社会制度、可用技术；
- `world_system`：种族/组织、层级、法律、仪式、能力与禁忌；
- `address_and_voice`：称谓、亲疏、权力语气、角色个人口头禅；
- `communication_channels`：面对面、电话、短信、社媒、书信、心灵链接等渠道的允许范围；
- `locked_terms`：必须保留、固定翻译和禁用译法；
- `evidence`：Project Bible、客户样例或团队资料的具体出处；
- `conflicts`：源稿与世界观/年代矛盾，转交修订流程处理。

## 北美超自然题材起始规则

只有项目确实采用相应体系时才加载对应段落。

### 狼人

- 常见体系词包括 `pack`、`packhouse`、`Alpha`、`Luna`、`Beta`、`Omega`、`rogue`、`mate bond`、`rejection`、`mind-link`、`inner wolf` 和 `Moon Goddess`。以项目术语表决定是否保留英文或双语处理。
- 常见称谓有 `Alpha [name]`、`Luna [name]`、`my mate`、`my Luna`、`little wolf`、`pup`；不要自动套用中式“大人/尊上”。
- `my wolf` 是人/狼双意识表达，不应误译成普通宠物或比喻。
- 现代背景允许手机，但 pack 事务常用 `mind-link`，正式挑战和仪式通常当面完成。翻译短信前先检查渠道是否合理。
- 项目锁定的 rejection/acceptance/claiming 誓词属于签名台词，必须逐版本对账。

### 黑手党

- 先确认 Italian/Cosa Nostra、Bratva、Cartel 或项目自定义分支，不得混用文化标记。
- `Don [surname]`、`Boss`、`Capo`、`famiglia`、`omertà` 等需进入术语表。意大利语通常只作点缀，不把整段英文对白无依据改成意大利语。
- 重要事务常当面谈，可能使用烧毁字条或一次性手机；不应在普通短信中直白讨论机密。需要保留短信时，检查是否应更短、更隐晦，并将结构冲突交给修订流程批准。

### 吸血鬼

- 常见词包括 `sire/maker`、`sire bond`、`fledgling`、`turning`、`first feeding`、`compulsion`、`daylight ring` 和 `vervain`。
- 古老角色常用完整句式、旧式礼貌和较少现代俚语；若误用现代俚语是刻意喜剧点，要保留“用错”的功能，不能润色成完全现代。
- 角色年代感应体现在对白、信件与短信习惯中，不只体现在名词。

### 龙族

- 区分 dragon shifter、rider/学院体系与项目自定义体系。常见词包括 `Dragon King/Queen`、`hoard`、`fireless`、`reverse scale`、`bond`、`relic`、`signet`、`my treasure`。
- `hoard`、`reverse scale` 等可能同时承担世界机制与情感隐喻；不要只按字典义翻译。

### 人鱼 / 塞壬

- Mermaid 与 Siren 的观众预期不同，先锁定物种体系。常见词包括 `pod`、`sea kingdom`、`trident`、`sea witch`、`voice trade`、`true love's kiss`、`siren song`、`moon pool`、`selkie`、`scales` 和 `land-walking`。
- 歌声、失声、名字和书面文字可能是能力机制；短信或字幕处理不能破坏“声音是否存在/谁能听见”的剧情条件。

## 冲突处理

发现不合年代的手机、不合组织规则的短信、违反能力限制的沟通或互相冲突的术语时：

1. 保留源文，不在翻译轮擅改母版；
2. 在 QA 报告标出场次、原文、规则证据和影响；
3. 提交“源稿修订建议”或“批准例外”供人工决定；
4. 批准后再更新 Project Bible/锁定项和译文。
